📖 How to Use

Create gap-fill listening exercises in 5 simple steps.

1. Add a video

  • Upload — drag/drop or click. Any common format works.
  • Link — paste a YouTube URL or direct .mp4 link.

2. Get transcript

Click ✨ Auto-Transcribe for AI-generated timestamps, or type your own text.

3. Build sections

Click 🤖 AI Generate Gaps to auto-create exercise sections, or add them manually.

  • Use _ in the text for each gap.
  • Comma-separate the answers.
  • Set start/end times to control which part of the video plays for each section.

4. Preview & share

  1. Preview — see what students will see.
  2. Generate Link — copy and share with students. No signup required.
  3. Download as JSON — keep a backup file.
💡 Cloud uploads take 1-3 minutes to process before you can use them.
⚠️ Local files are preview-only — students can't see them. Use Cloud Upload for sharing.
